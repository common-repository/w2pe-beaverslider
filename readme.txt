w2pe-beaverslider is one of simple image slider made for wordpress blog or site. 

== Description ==

"w2pe-beaverslider" you can download it for you next image slider into wordpress blog or site. It base on "Beaverslider". 

**Detailed features:** 

* Use it as your home page slider, into post or page.
* User friendly.
* You can use it into post and page by shortcode <code>[w2pe_bSlider] </code> into page or page and into template file using <code> <?php if (function_exists('w2pe_bslider')) { w2pe_bslider(); } ?></code>

== Installation ==

1. Upload the entire `w2pe-beaverslider` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'w2pe-beaverslider' menu in your WordPress admin panel.

For basic usage, you can also have a look at the [plugin homepage](http://www.wppluginexpert.com/) plugin section.
